# About
Geopelia is Twitter Client for Windows 10 Platform.  
It is currently in development.

Geopelia（ジオペリア）は Windows 10 プラットフォーム用に開発している Twitter クライアントです。  
Geopelia は2016年中のリリースを目標としています。
